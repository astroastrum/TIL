## Grid System

* 그리드 컨테이너 (Grid Container)

  * 부모 요소 

* 그리드 아이템 (Grid Item)

  * 자식 요소 

* Grid System 어떻게 시작하는가?

  ```css
  .container {
      display: grid;
  }
  ```
