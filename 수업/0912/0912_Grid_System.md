## Grid System

* Offset

  * 어떠한 **차이**, 기준이 되는 것에서 벗어난 차이
    * 주소 A가 50번지를 의미할 때 A+9의 9는 Offset을 의미합니다. 

* @media

  * Media Query

    * CSS는 선택을 합니다. 이와 같이, Media Query는 디자인을 원하는 요소에 입히고 싶을 때 작동합니다.

    * Media Query 구조

      ```css
      @media (조건) {
          스타일
      }
      ```
