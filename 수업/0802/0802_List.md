## Range 

```python

range()
# 파이썬 내장함수 
# range()의 역할은 순서가 있는 또는 순차적인 정수 리스트를 만듭니다

range3 = range(30)
# 0 이상 30 미만까지
# range 객체를 주는데 range 객체는 리스트라고 생각하시는 분들 많은데 그렇지 않습니다

print(list(range3))
# range 객체를 바로 print할 수 없고 print하기 위해서는 실제 값을 출력 해야하는데 
# list를 통해서 형변환을 해줘야 합니다


count = 0
for i in range(20):
    count += (i+1)
    
# 위 코드는 1부터 20까지 정수를 add.

```
