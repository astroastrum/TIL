## if문

### if~else

```python
참인지 거짓인지 구별하는 if문
참이면 실행하고
거짓이면 실행하지 않습니다

a = 7
b = 10
if a > b:
    print('a는 b보다 뛰어납니다')
else:
    print('a는 b보다 뛰어나지 않아요')

```



### if~elif

```python

a = 7
b = 10

if a == b:
    print('a는 b와 똑같습니다')
elif a > b:
    print('a는 b를 능가합니다')
else:
    print('a is smaller than b')
    
if 조건 1이 true이면 실행합니다.
elif 조건 2가 true이면 실행합니다.
위의 두 조건이 거짓이면 else의 실행 코드를 출력합니다.
```
