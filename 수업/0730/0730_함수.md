### result의 필요성

```python
# 문제 예시
주어진 문자열 word가 주어질 때, 해당 단어에서 'p'를 모두 제거한 결과 출력

word = 'peach'

for char in 'peach':
    if char != 'p':
        print(char)
        
이렇게 출력하면 출력이 한 줄로 되지 않습니다. 그래서 result라고 하는 문자열을 만들어서 해결할 수 있습니다.

word = 'peach'
result = ''
for char in 'peach':
    if char != 'p':
    result += char

print(result)

```
