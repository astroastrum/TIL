## Literal 리터럴

* Literal 리터럴

  * 값 자체를 뜻함

* Constant 상수

  * 공간
    * 값을 한번만 담을 수 있는

* 문자 리터럴

  * 작은따옴표 안에 문자 하나만 담은 것

    ```java
    char c = 'S';
    ```

  * 작은따옴표 안에 반드시 하나의 문자가 존재해야함

* 문자열 리터럴

  * 큰따옴표 안에 두 문자 이상을 담은 것

    ```java
    String use = "Literal";
    ```

  * String타입 사용

    * 언제?
      * 여러 문자를 담을 경우
