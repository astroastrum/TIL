## switch문

* 조건식을 계산한 후 계산된 결과와 같은 case문으로 이동합니다. 일치하는 case문의 문장을 수행하고 break문 또는 switch문의 끝을 마주치면 switch문에서 해방됩니다.

  ```java
  switch (조건식) {
      case 값1 :
          // 조건식의 결과와 값1이 똑같으면 실행되는 문장
          break;
      case 값2 :
          // 문장
          break;
      default :
          // 동일한 결과가 없으면 실행되는 문장
  }
  ```
